# DataBase-Lab1
Первая лабораторная по базам данных(структура БД) 
# 45. Социальная сеть, анализ постов и активности
🏷️ Сущности: Пользователи (логин, дата регистрации), посты (ID, дата публикации, текст), реакции (тип - лайк/комментарий/репост), подписки (кто на кого подписан).

🔄 Процессы: Пользователи публикуют посты, другие пользователи оставляют реакции.

📄 Выходные документы:

- 📋 Для заданного пользователя выдать список его постов, отсортированный по убыванию суммарного количества лайков и комментариев за последнюю неделю.
    
- 🏆 Выдать список самых популярных постов (по количеству реакций) в заданной тематике (определяется по ключевым словам) за месяц.
